# Multi-Answer Generation

## Question 1: What is force majeure?

### Observation

All three generated answers communicated the same meaning. The wording varied slightly, but each answer described force majeure as events beyond a party's reasonable control that excuse delays or failures in performance. All answers referenced natural disasters, war, government actions, notification obligations, and termination rights after 30 days.

### Result

High agreement.

## Question 2: Can termination occur immediately?

### Observation

All three answers identified bankruptcy or insolvency as a circumstance where immediate termination is allowed. One answer included additional information about force majeure termination, but the primary interpretation remained consistent across all answers.

### Result

High agreement.

## Question 3: Can an employee work another job?

### Observation

The retrieval system did not find any relevant information about outside employment. All three answers correctly stated that the provided context did not contain enough information to answer the question.

### Result

High agreement.

## Key Learning

Generating multiple answers reveals whether the model consistently interprets the retrieved context. If all answers communicate the same meaning, confidence should be higher. If answers disagree, confidence should be lower. This forms the foundation of semantic entropy.
